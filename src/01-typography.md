| Token | Value | Purpose|
|:---|:---:|:---|
|`--font-default-size`| `14px` | Default font size for elements like `html`. |
|`--text-size-extra-large`| `3rem` | Used for big titles and display heros. |
|`--text-size-large`| `2.5rem` | Used for secondary section and character titles |
